## Troubleshooting

### Login Issues

**Problem:** Can't login to account

**Solutions:**
1. Check email/password spelling
2. Use **"Forgot Password"** to reset
3. Clear browser cache and cookies
4. Try different browser
5. Check if account is verified

### Email Not Received

**Problem:** Verification or reset email not received

**Solutions:**
1. Check spam/junk folder
2. Add noreply@keohams.com to contacts
3. Wait a few minutes (may be delayed)
4. Click **"Resend Email"**
5. Contact support if still not received

### Payment Failed

**Problem:** Payment not going through

**Solutions:**
1. Check card details are correct
2. Ensure sufficient funds
3. Verify card is enabled for online transactions
4. Try different payment method
5. Contact your bank
6. Contact support with screenshot

### Order Not Showing

**Problem:** Can't find order after payment

**Solutions:**
1. Check **"Order History"**
2. Wait a few minutes (system may be processing)
3. Check confirmation email
4. Verify payment was successful
5. Contact support with transaction reference

### Product Out of Stock

**Problem:** Want to buy out of stock item

**Solutions:**
1. Click **"Notify When Available"**
2. Check back regularly
3. Look for similar products
4. Contact support for restock information

### Delivery Issues

**Problem:** Order not delivered on time

**Solutions:**
1. Check order status
2. Track shipment with tracking number
3. Contact courier company
4. Contact support if overdue

### Website Loading Slowly

**Problem:** Website loading too slow

**Solutions:**
1. Check internet connection
2. Clear browser cache
3. Disable browser extensions
4. Try different browser
5. Use mobile app (if available)

### Can't Upload Documents

**Problem:** Document upload failing

**Solutions:**
1. Check file size (max 5MB)
2. Use supported formats (JPG, PNG, PDF)
3. Compress large files
4. Try different browser
5. Check internet connection
